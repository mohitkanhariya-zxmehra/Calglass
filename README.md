# Calglass

A glass-style calculator with welcome animation, fluid background, memory (M+), sound, and haptic feedback.  

## How to run

1. Open `index.html` in a browser.  
2. Or host on GitHub Pages:

   - Push this repository to GitHub.  
   - Go to Settings → Pages → Select `main` branch → Save.  
   - Visit `https://<username>.github.io/Calglass/`

## Features

- Glass effect with blur and fluid background  
- Memory (M+) button  
- Haptic feedback on mobile  
- Sound click option  
- Responsive layout for mobile and desktop
